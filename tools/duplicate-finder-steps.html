How to Use

1
Upload Your Dataset
Upload the Excel or CSV file you want to check for duplicates.
2
Select Duplicate Key Columns
Choose which columns define a duplicate (e.g., Email only, or First Name + Last Name + Email together).
3
Review Duplicate Report
Datacorex shows you all duplicate groups before removal so you can review what will be affected.
4
Configure Keep Rule
Choose whether to keep the first occurrence, the last occurrence, or manually select which rows to keep.
5
Remove & Download
Click Remove Duplicates. Download your clean deduplicated Excel file.

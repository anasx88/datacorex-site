How to Use

1
Upload Your Master File
Upload the large Excel or CSV file you want to split into separate files.
2
Choose Split Column
Select the column whose unique values will determine how the file is split (e.g., Department, Region).
3
Configure File Naming
Choose how the output files should be named — by column value or with a custom prefix.
4
Run the Split
Click Split. Datacorex processes every row and creates the output files instantly.
5
Download ZIP Archive
Download a ZIP file containing all the split Excel files, ready to distribute.

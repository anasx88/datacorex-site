WORKFLOW

Input → Process → Output

Visual workflow for Duplicate Finder — upload, detect, review, and export.

INPUT
Excel / CSV Dataset
Upload your dataset. It can contain any number of rows and columns.
PROCESS
Column-Based Duplicate Detection
Datacorex scans all rows and groups duplicates based on your chosen columns. Each group is flagged for review.
OUTPUT
Clean Deduplicated File
Download a clean Excel file with duplicates removed, plus a separate report of all duplicates that were found.
INPUT
Real example
Employee_List.xlsx with repeated Employee ID or Email values
OUTPUT
What you get
Duplicate groups identified, a cleaned dataset, and a review file for verification.

THE TOOL

What This Tool Does

The Merge Columns tool is a no-formula alternative to VLOOKUP and INDEX/MATCH. It matches rows between two files using a shared key column, then pulls selected columns from the second file into your main dataset. The result is a single unified Excel file with all the data you need in one place — no formulas, no errors, no broken references.

FAQ

Frequently Asked Questions

Is this the same as VLOOKUP?
It achieves the same result as VLOOKUP but is much simpler — no formulas, no syntax errors, and no broken references when data changes.
What if a key value exists in the main file but not the lookup file?
Rows without a match will have blank values in the added columns. You can optionally include or exclude unmatched rows.
Can I add columns from multiple lookup files?
Run the tool multiple times — once per lookup file. Each run adds new columns to your growing dataset.
Does it handle duplicate key values?
Yes. If the key column has duplicate values in the lookup file, Datacorex uses the first match by default and notes duplicates in the output.

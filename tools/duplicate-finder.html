THE TOOL

What This Tool Does

The Duplicate Finder tool scans your Excel or CSV dataset for duplicate rows based on any column or combination of columns you choose. It identifies exact duplicates and near-duplicates, shows you what was found, and lets you remove them with a single click — keeping either the first or last occurrence of each record. The output is a clean, deduplicated Excel file with a summary of what was removed.

FAQ

Frequently Asked Questions

Can I define what counts as a duplicate?
Yes. You choose which columns to use as the duplicate key. A duplicate is any row where all selected columns match another row.
What if I want to keep the most recent duplicate instead of the first?
You can choose to keep the first or last occurrence of each duplicate group before removing them.
Will I see the duplicates before they're removed?
Yes. Datacorex shows you a full duplicate report first, so you can review everything before deciding to remove them.
Can I remove duplicates based on just one column like Email?
Yes. Select only the Email column as your key, and any rows sharing the same email will be treated as duplicates.

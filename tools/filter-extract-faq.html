FAQ

Frequently Asked Questions

How many filter rules can I set at once?
You can define as many rules as you need — one per column or multiple rules on the same column.
Can I filter by date ranges?
Yes. Date columns support before, after, between, and exact match conditions.
What condition types are supported?
Equals, not equals, contains, starts with, ends with, greater than, less than, between, is blank, and is not blank.
Can I extract to a separate file without modifying the original?
Yes — Filter & Extract always creates a new output file. Your original data is never modified.

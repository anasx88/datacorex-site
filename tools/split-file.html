THE TOOL

What This Tool Does

The Split File tool takes a single Excel or CSV file and automatically divides it into multiple separate output files — one file per unique value in a column you choose. Split a company-wide dataset by department, a national report by region, an annual log by month, or a product catalog by category. Each output file contains only the rows for that value, ready to distribute or process independently.

FAQ

Frequently Asked Questions

How many output files can the tool create?
As many as there are unique values in your chosen column. Splitting by a column with 50 unique values creates 50 files.
Will each output file keep the original headers?
Yes. Every output file includes the original column headers from the source file.
What if the column I split by has blank values?
Rows with blank values are grouped into a separate 'Blank' file, so no data is lost.
Can I split by multiple columns at once?
Currently the tool splits by one column at a time. For multi-column splits, run the tool twice on the output files.

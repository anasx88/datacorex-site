FAQ

Frequently Asked Questions

Does the comparison require the same row order?
No. Datacorex matches rows by a key column (like an ID or name), so the row order doesn't need to be the same.
What if one file has extra columns?
Extra columns in either file are flagged as Added or Removed in the report. The comparison still works for matching columns.
Can I compare files with 100,000+ rows?
Yes. The tool handles large files efficiently. Performance depends on your device's available memory.
What does the output report contain?
A Run Summary sheet, a Differences sheet showing changed cells side-by-side, an Added Rows sheet, and a Deleted Rows sheet.
